模块 B：金融文本智能项目

用受控 Vibe Coding 分析巨潮公告

作者

汪小圈、邵颖慧

发布于

2026年5月6日

本讲义用于《数据挖掘与机器学习》课程 Week 11–16，即后半程的金融文本智能项目模块。

模块 B 的核心任务是：每个小组从巨潮资讯网公开公告中选择一个金融文本分析问题，使用受控 vibe coding 搭建一个小型、可复现、可检查的文本智能系统。最终成果不是一份单纯的文字报告，而是一套包含数据、代码、schema、prompt、解析结果、抽取结果、评估报告和展示材料的项目。
