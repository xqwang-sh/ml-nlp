# 选题 Rubric

教师给出的题目只是参考，学生可以自拟题目，但必须满足以下条件。

| 维度 | 通过标准 |
|---|---|
| 巨潮数据可获得 | 能在巨潮找到足够样本 |
| 金融问题清楚 | 有风险、合规、治理、业绩、分红、市场沟通等金融含义 |
| 字段可结构化 | 能写成 JSON / CSV，能判断对错 |
| 证据可追溯 | 能提供公告标题、日期、页码、原文片段 |
| 6 周可完成 | 工作量合理，不依赖不可控数据 |
| 难度可声明 | 能说明为何是 0.9 / 1.0 / 1.1 |

## 红灯信号

- 数据不是巨潮来源；
- 只有“总结公告”，没有结构化字段；
- 字段无法判断对错；
- 需要不可获得的数据；
- 6 周内无法完成；
- 选题过大且没有降级方案。
