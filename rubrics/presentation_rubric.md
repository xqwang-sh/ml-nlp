# 展示与答辩 Rubric

| 维度 | 分值 | 评价标准 |
|---|---:|---|
| 金融问题表达 | 3 | 能讲清为什么该公告任务有价值 |
| 方法链路表达 | 3 | 能讲清数据、API、Schema、MinerU、Section、Workflow 如何串联 |
| Demo 与证据链 | 4 | 能展示一个 PDF 到结构化结果的过程；字段有证据文本和页码 |
| 评估与局限 | 3 | 能说明系统好坏、失败案例和下一步优化 |
| Vibe Coding 反思 | 2 | 能说明 AI 帮了什么、错在哪里、学生如何验证 |

## 答辩重点

- 是否能追溯一条结果；
- 是否能解释一个错误；
- 是否能说明 AI coding agent 的作用和边界；
- 是否能说明难度档位是否合理。
